# Customer Churn Analysis

## 📊 Overview

This project analyzes customer data to understand churn behavior and retention patterns.

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib

## 🔍 Analysis Done

* Churn rate calculation
* Churn by gender and contract type
* Customer behavior analysis

## 📈 Key Insights

* Identified customers likely to churn
* Contract type impacts churn
* Customer tenure affects retention

## 🚀 Conclusion

This analysis helps businesses reduce churn and improve customer retention strategies.
